# Match PWA (starter)

## Install
```bash
npm i
npm run dev
```

## Configure backend
Create `.env` file (or copy from `.env.example`) and set:
- `VITE_API_BASE`

## Notes
- PWA is enabled via `vite-plugin-pwa` (autoUpdate).
- This structure is ready to grow by adding pages in `src/pages/*` and shared UI in `src/shared/ui`.
